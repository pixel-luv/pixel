### [요시조]

1. 요시조는 본진 1라인 진영 후냐 배치 후 각인타워로 이동
2. 요시조는 후냐 배치를 마친 후 각인 타워 및 본진 1라인 구역 정찰 및 상대 견제
3. 포털을 이용하여 1라인 진입하는 상대견제
4. 각인타워쪽에서 확인되는 상대 바리케이드 제거
